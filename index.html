프로젝트 소개

FilmPlay

영상 제작사를 위한 인스타 포맷의 영상 포트폴리오 어플리케이션입니다.
자체 제작한 영상물 홍보 및 다른 제작사의 영상을 홍보할 수 있습니다.

개발 환경

개발 기간 : 2023.11.13. ~ 2023.11.27.

Front-End : React, Styled Components
Back-End : Firebase
상태 관리 : Zustand
웹 배포 : Netlify
웹 앱 배포 : PWA (Progressive Web App)
